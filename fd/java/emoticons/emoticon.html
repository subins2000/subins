Usage

To replace the text of emoticons just make the container of class="posted". Pretty easy, right? :)

Known Bugs

Bug B0001 is caused due to the simple(naive) replacement method. There should be a traversal of the DOM or a regular expression replacement in order to get better results, but that would be rather slow. See the source code to understand the problem.
  • B0001: Emoticons in text only elements(image alternate text, input button, etc), will be replaced by an ugly html code
